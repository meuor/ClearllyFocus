# 🌀 ClearllyFocus V3.01
> **The Ultimate Neuro-Sync Environment for Deep Work.**

**ClearllyFocus V3.01** is a high-performance web application engineered to trigger and maintain the **Flow State**. Unlike standard "lo-fi" players, this app uses the **Web Audio API** to synthesize sound waves in real-time, providing a seamless, gapless acoustic blanket that blocks out distractions and anchors your focus.

---

## 🚀 The Flow-State Toolkit

### 🎧 Procedural Audio Engine
Forget MP3 loops. ClearllyFocus generates sound mathematically:
* **Brown Noise:** Deep, low-frequency rumble for heavy-duty focus.
* **White Noise:** Balanced static to mask chaotic environments.
* **Pink Noise:** Softened frequencies for long-term reading and clarity.
* **Alpha/Beta Overlays:** Integrated Binaural Beats (10Hz/20Hz) to synchronize brainwaves for peak cognitive performance.

### 🎯 Psychological Anchors
* **The One-Task Locker:** A high-visibility focus banner that forces you to commit to a single objective.
* **Minimalist UI:** A "Zen-Mode" Glassmorphic design that reduces visual noise and cognitive load.
* **Vitals Reset:** A built-in visual pacer for **4-7-8 Breathing** to transition your nervous system from "Stress" to "Flow."
