# ⚔️ ClearllyFocus V4.05 - Flow State Engine
**Bilingual Productivity & Spirituality Platform**

تطبيق "ClearllyFocus" هو محركك الشخصي للوصول لحالة التركيز القصوى (Flow State)، يدمج بين تقنيات الإنتاجية الحديثة والجانب الروحاني بنظام ألعاب (Gamification) فريد.

---

## 🚀 التحديثات الأخيرة (V4.05)
تم تحويل التطبيق من مجرد واجهة ثابتة إلى تطبيق سحابي كامل (Full-Stack) مع الميزات التالية:
* **Firebase Integration**: ربط كامل مع قاعدة بيانات Firestore لحفظ تقدمك وساعات تركيزك سحابياً.
* **Google Auth**: تسجيل دخول آمن باستخدام حساب جوجل لمزامنة بياناتك عبر المتصفحات.
* **Optimized Build**: استخدام Vite + TypeScript لضمان سرعة تحميل فائقة واستقرار الكود.
* **PWA Ready**: التطبيق جاهز للتثبيت على الموبايل (iOS/Android) ليعمل كتطبيق أصلي.

---

## ✨ المميزات الرئيسية

### 1. محرك التركيز (Focus Engine)
* **Pomodoro Timer**: فترات قابلة للتخصيص (25/5, 50/10, 45/15).
* **Immersive Moods**: 5 خلفيات تفاعلية (Rain, Fire, Forest, Cafe, Street) مع تدرجات لونية مريحة للعين.
* **Audio Engine**: أصوات نويز (White, Brown, Pink) وموجات Binaural Beats (Alpha & Beta) لتحسين كفاءة الدماغ.

### 2. طريق المحارب (Warrior's Path) ⚔️
نظام "ليفل" يحول ساعات المذاكرة إلى أسلحة ورتب:
| الساعات | السلاح | الأيقونة |
| :--- | :--- | :--- |
| 1h | Dagger | ⚔️ |
| 10h | Scimitar | 🗡️ |
| 50h | Knight's Shield | 🛡️ |
| 100h | Great Bow | 🏹 |
| 500h | King's Crown | 👑 |

### 3. القسم الروحاني (Spirituality)
* **Prayer Times**: عرض أوقات الصلاة تلقائياً باستخدام Aladhan API بناءً على موقعك.
* **Spiritual Checklist**: متابعة الصلوات، الأذكار، وقراءة القرآن.
* **Wisdom Feed**: اقتباسات متجددة من القرآن والأحاديث النبوية.

---

## 🛠️ البناء التقني (Tech Stack)
* **Frontend**: React + TypeScript + Tailwind CSS.
* **Backend**: Firebase (Authentication & Firestore).
* **Deployment**: GitHub Pages.
* **Build Tool**: Vite.
